マージする時はmainにあるステージングか未ステージの.kamui/tasks-state.jsonのmainの方を先にコミットしてからマージ作業すること

以下の手順でmainブランチへのマージを実行してください：

1. 現在の作業をコミット
   - git add . でステージング
   - git commit -m "適切なコミットメッセージ" でコミット（メッセージは作業内容に応じて記述）
- 現在のディレクトリを確認

2. リポジトリのルートディレクトリに移動
   - cd /Users/apple/Desktop/AI アプリ/NodeVision Editor _V2

3. mainブランチにチェックアウトしてマージ
   - git checkout main
   - git merge --no-ff task/元々いたディレクトリのタスクID

実際のコマンド例：
git add . && git commit -m "適切なコミットメッセージ" && /Users/apple/Desktop/AI アプリ/NodeVision Editor _V2 && git checkout main && git merge --no-ff task/元々いたディレクトリのタスクID
