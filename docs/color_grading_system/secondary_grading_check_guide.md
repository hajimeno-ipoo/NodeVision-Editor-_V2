# セカンダリーグレーディング確認ガイド（ノードに出ている項目だけで説明）

「つまみ」＝ノード内のスライダーの丸いポッチ。左右に動かすと数字が変わり、選択範囲や効果が変わる。

## 0. 配線
`Load Image` → `Secondary Grading` → `Media Preview` をつなぐ。画像が出れば準備OK。

## 1. HSL Qualifier（Hue / Saturation / Luminance の3セット）
どのセットも **Center / Width / Softness** の3本スライダー。  
必ず `Show Mask` を ON（白＝選ぶ範囲）。これを見ながらテストする。

### Hue
- **Center (0〜360)**: どの色を選ぶか。ざっくり 0=赤,120=緑,240=青。  
  - テスト: Center を動かすと白い範囲が赤→黄→緑→青…と移動し、画像でもその色の部分だけが後の補正で変わること。  
  - NG: Center を動かしても白い範囲が変わらない／全体が一緒に変わる。
- **Width**: 選ぶ色の幅。  
  - テスト: 広げると白い範囲が太くなり、狭めると細くなる。  
  - NG: 動かしても白い範囲の広さがほぼ同じ。
- **Softness**: 境目のぼかし。  
  - テスト: 上げると白黒の境目がふわっと、下げるとくっきり。  
  - NG: 境目の見え方が変わらない。

### Saturation（色の濃さで選ぶ）
- Center: 濃い色／薄い色のどちらを中心に選ぶか。  
- Width/Softness: Hue と同じ動作。  
- テスト: 濃い色の服・薄い色の肌で、白い範囲が切り替わるか。

### Luminance（明るさで選ぶ）
- Center: 明るい所／暗い所のどちらを中心に選ぶか。  
- Width/Softness: Hue と同じ動作。  
- テスト: 明るい空と暗い影で白い範囲が切り替わるか。

### Invert Selection
- 白黒を反転。反転後、さっきとは逆の場所が白くなるか確認。

## 2. Correction（選ばれた白い部分だけが変わるかを見る）
- **Saturation**: 白い部分の色の濃さを上下。  
- **Hue Shift**: 白い部分の色味を回す（赤→黄→緑…）。  
- **Brightness**: 白い部分の明るさを上下。  
- **Intensity**: 全体の効き目（0で効果なし、1で元どおり）。

すべて「白い部分だけが変わること」を確認。全体が一緒に変わるならNG。

## 3. レイヤー
- `+ Add` でレイヤー追加。タブを切り替えると各レイヤーは独立しているか確認。  
- NG: Layer 1 を動かすと Layer 2 まで同じ変化をする。

## 4. すぐ試せるOK/NG例
- 空だけ濃く：Hue Center を青付近、Width をやや狭く、下の Saturation を少し上げる。  
  - OK: 空だけ濃く、肌や服はほぼそのまま。  
  - NG: 肌まで青くなる、全体が濃くなる。
- 肌だけ明るく：Hue Center を肌色付近、Width を狭め、Brightness を少し上げる。  
  - OK: 肌が明るく、背景は大きく変わらない。  
  - NG: 背景も一緒に明るくなる。

## 5. 動画でも同じテスト
- `Load Video` → `Secondary` → `Media Preview`。再生したままスライダーを動かし、選んだ白い範囲だけが変わるか見る。  
- NG: 再生が止まる、真っ黒/真っ白になる、全体が同じように変色する。

## 6. よくあるつまずき
- 全体が動く → Hue/Sat/Lum の Width が広すぎ。狭める。  
- マスクが真っ白/真っ黒 → Center/Width を調整し直す。  
- 反映しない → ノード配線を確認、再起動。

これで「ノードにあるスライダーを動かすと何がどう変わるか」を確かめやすくなるよ！ 🎨
