# セカンダリーグレーディング動作確認ガイド（かんたん版）

こんにちは！小学生でもできるチェック手順だよ。ゆっくりやってね〜 ✨

## 0. じゅんび
- アプリを起動する（NodeVision をいつも通り開くだけ）。
- もしGPUがこけてプレビュー真っ黒になったら、一回アプリを閉じてもう一度開こう。それでもダメならパソコン再起動だよ。

## 1. 画像で試す
1. 左のノードリストから「Load Image」を置く。
2. 次に「Secondary Grading」を置いて、Load Image と線でつなぐ。
3. 「Media Preview」を置いて、Secondary Grading とつなぐ。
4. Secondary のスライダーを動かす。  
   - 動かしている間はすばやいプレビュー（低画質）。  
   - 手を放すときれいなプレビュー（高画質）になるよ。
5. 「Show Mask」をオンにすると白黒マスクが見えるよ。

## 2. レイヤーをふやす
1. Secondary ノードの上にあるタブで「＋追加」を押す。  
   新しいレイヤーができるよ。
2. それぞれのレイヤーで色を変えてみる。  
   同じ場所でも色が重なるから、変化が合わさって見えるよ。
3. いらないレイヤーは「－削除」で消せる。
4. 失敗したら「レイヤーリセット」か「全部リセット」を押せばもとに戻る。

## 3. Intensity で強さを変える
1. 「Intensity」スライダーを0〜1で動かす。  
   0に近いほど弱く、1で全力だよ。
2. いっぱいレイヤーがあるときは、アクティブなタブのレイヤーだけが動くよ。

## 4. 動画で試す
1. 「Load Video」を置いて、Secondary につなぐ → Media Preview へ。  
   （動画が再生できるファイルを選んでね）
2. スライダーを動かすと、動画にもちゃんと反映されるか確認。  
   重かったら一度待って、高画質反映後にもう一度見るといいよ。

## 5. こまったとき
- プレビューが動かないとき：  
  - もう一度ノードをつなぎ直す。  
  - アプリを再起動する。  
- GPUエラーっぽいとき：  
  - アプリ再起動 → だめならPC再起動。  
  - まだダメなら大人に相談してね。

おつかれさま！これでセカンダリーグレーディングのチェックはばっちりだよ 🎉
